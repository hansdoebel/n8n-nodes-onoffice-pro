# Introduction
	

	

		
Welcome to the API documentation of onOffice enterprise. The data of many modules (addresses, properties, tasks, calendar etc.) can be read and written via API.

For example, you can display the current real estate offers from onOffice on your website, create your own useful apps with a data connection to onOffice or save your data from onOffice.

The initial connection to the onOffice API and the authentication process are described in detail under &#8220;[First steps / API connection](https://apidoc.onoffice.de/erste-schritte/)&#8220;.

The request URL for the stable API version, which is updated monthly, is [https://api.onoffice.de/api/stable/api.php](https://api.onoffice.de/api/stable/api.php). The request URL for the latest API version is [https://api.onoffice.de/api/latest/api.php](https://api.onoffice.de/api/latest/api.php).

The onOffice SDK is available on [GitHub](https://github.com/onOfficeGmbH/sdk) and can be downloaded there.

The onOffice API is available to all enterprise customers as a paid module. For the exact conditions please contact your sales contact at onOffice.
