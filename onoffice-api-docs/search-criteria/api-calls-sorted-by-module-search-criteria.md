# Search Criteria
	

	

		
Links with other modules can be read and written via the [relations](https://apidoc.onoffice.de/api-calls-sorted-by-module/relations/) calls. For search criteria, only addresses can be linked to search criteria.

Relationtype: `'urn:onoffice-en-ns:smart:2.5:relationTypes:address:searchcriteria' searchcriteria for address (address = parent record, searchcriteria = child record)`

[Online help about search criteria](https://de.enterprisehilfebeta.onoffice.com/help_entries/adressdetailansicht/immosuche-ueberblick/immosuche-details/)
